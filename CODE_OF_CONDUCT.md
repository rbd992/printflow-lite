# Code of Conduct

## Our Pledge

PrintFlow Lite is a project built by a maker for makers. Everyone who participates — through issues, pull requests, discussions, or comments — is welcome here regardless of experience level, background, or how many 3D printers they own.

## Our Standards

**Be kind.** This is a small open source project built by one person in their spare time. Treat everyone with the same respect you'd want in return.

**Be constructive.** Criticism is welcome — cruelty isn't. If something is broken or poorly designed, explain why and suggest how it could be better.

**Be patient.** Issues and PRs are reviewed as time allows. This isn't a company with a support team.

**Be honest.** If you find a security vulnerability, please report it privately via [GitHub Security Advisories](https://github.com/rbd992/printflow-lite/security/advisories/new) rather than opening a public issue.

## Enforcement

Behaviour that violates this code of conduct — harassment, personal attacks, or sustained disruption — will result in removal from the project. This is a place for people who want to build something useful together.
